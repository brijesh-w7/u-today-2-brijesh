# u-today-2-brijesh



> The player is shown as a green rectangle and is controlled using a virtual joystick.
</br><br/>
> The table, refrigerator, trash, and other objects are shown as red rectangles.<br/><br/>
> I have implemented a toast feature to show short messages that disappear after 1–2 seconds.<br/><br/>
> The score system is implemented as mentioned in the document.<br/><br/>
> A loader is added for processing ingredients like meat and vegetables.<br/><br/>
> I have added basic sound effects. The same click sound is used everywhere, and it can be changed from the SoundManager game object. Separate fields are created for each sound.<br/><br/>
> The stove feature has two slots implemented.<br/><br/>
> The trash feature is also implemented.<br/><br/>
> The objects the player can carry are 3D objects. Their material changes based on their state (raw, processed, or cooked).<br/><br/>
> UI screens implemented: Main Menu Panel, Pause Panel, Game Over Panel, and Order Panel for each window with received ingredients.<br/><br/>
> A safe area fix is added for devices with a top notch.<br/><br/>
> The UI is compatible with all screen resolutions.<br/><br/>
